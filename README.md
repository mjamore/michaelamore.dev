# michaelamore.dev
The source code for my personal website.

## How to run locally:
- From the root of the project run the following commands:
	1. `npm install`
	2. `npm start`

## How to Deploy:
- From the root of the project run the following commands:
	1. `npm run build`
	2. some deployment command here...
